# Task: Difficulty Selector Web App

Build a polished, responsive, frontend-only web app that lets a player choose a game difficulty level before starting a session.

## Requirements

- Hero title: **Choose Your Difficulty**
- Display five difficulty levels: Very Easy, Easy, Medium, Hard, Insane
- Each card shows a short description and balancing values:
  - Player Health
  - Enemy Health
  - Enemy Damage
  - Loot / Resources
  - Enemy Spawn Rate
- Clear selected-state interaction (click to select, visual highlight)
- Data structured as a typed array — easy to extend
- Static / frontend only (no backend needed)
- Responsive layout (mobile + desktop)
