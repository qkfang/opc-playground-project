What we do

End-to-end autonomous systems research and development.

Autonomous Navigation

We develop perception-driven navigation stacks that let robots explore, map, and reliably traverse complex indoor and outdoor environments.

Manipulation & Grasping

Our robotic arms use vision-based grasp planning to handle diverse objects — from structured bin-picking to unstructured tabletop tasks.

Safety & Verification

Every system ships with hardware E-stops, software safety envelopes, and repeatable test suites so we can validate behavior before any real-world run.

Sensor Fusion

We fuse cameras, LiDAR, IMU, and wheel odometry through probabilistic filters to produce robust state estimates even under sensor degradation.

Simulation & Testing

Closed-loop simulation lets us run thousands of scenario variants overnight, catching edge cases before they reach hardware.

Open-Source Tooling

Design documents, build logs, ROS nodes, and experiment results are published openly so the community can build on our work.

Project highlights

Our current active platforms and experiments.

Rover-1 — Indoor SLAM

Differential-drive testbed for navigation and mapping in dynamic environments.

  • Sensors: stereo camera, IMU, LiDAR
  • Compute: edge GPU (NVIDIA Orin)
  • Stack: ROS 2, Nav2, RTAB-Map
  • Status: active testing
Docs (coming soon) Code (coming soon)

Arm-2 — Bin Picking

6-DoF arm with perception-driven grasping for tabletop manipulation tasks.

  • Payload: 1–2 kg
  • End-effector: parallel gripper
  • Stack: MoveIt 2, depth-based grasp planner
  • Status: integration testing
URDF (coming soon) ROS nodes (coming soon)

Drone-3 — Outdoor Nav

Vision-aided quadrotor for GPS-denied waypoint navigation.

  • Frame: 250 mm racing frame
  • Autopilot: PX4 + companion computer
  • Stack: VIO, custom planner
  • Status: hardware bring-up
